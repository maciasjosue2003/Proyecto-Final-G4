# BarberTurn - Sistema de Gestión de Citas y Turnos para Barbería

## Descripción del proyecto

BarberTurn es un prototipo académico desarrollado para organizar la gestión de citas y turnos en una barbería. 
El sistema permite registrar clientes, servicios, barberos y citas, con el objetivo de mejorar el control de la atención diaria.
Evitar problemas como turnos duplicados, pérdida de información o confusión en los horarios.

## Objetivo general

Desarrollar un prototipo funcional para la gestión básica de citas y turnos en una barbería.
Aplicando conceptos de Ingeniería de Software, levantamiento de requerimientos y planificación de proyectos.

## Funcionalidades iniciales

- Registrar clientes con sus datos básicos.
- Registrar los servicios ofrecidos por la barbería.
- Registrar barberos disponibles.
- Crear citas indicando cliente, servicio, barbero, fecha y hora.
- Consultar las citas registradas.
- Actualizar el estado de una cita, por ejemplo: pendiente, atendida o cancelada.
